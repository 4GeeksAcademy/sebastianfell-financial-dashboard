# Reglas de agentes

Este directorio contiene reglas operativas para trabajar en este repositorio.

## Objetivo
Definir reglas concretas, verificables y aplicables al flujo real del proyecto.

## Convenciones
- Una regla por archivo.
- Nombre del archivo con prefijo numerico y tema claro.
- Cada regla debe incluir:
	- Proposito
	- Alcance
	- Disparadores
	- Fuera de alcance
	- Checklist verificable
