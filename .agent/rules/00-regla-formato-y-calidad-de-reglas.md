# Regla: Formato y calidad de reglas

## Proposito
Evitar reglas ambiguas o de bajo valor operativo.

## Alcance
Aplica a toda regla nueva o modificada dentro de .agent/rules.

## Disparadores
- Se crea una regla nueva.
- Se edita una regla existente.

## Fuera de alcance
- Documentacion funcional del producto.
- README general del repositorio.

## Checklist verificable
- La regla incluye Proposito, Alcance, Disparadores, Fuera de alcance y Checklist.
- La regla tiene criterio verificable de cumplimiento.
- La regla cita al menos un archivo real del repo como contexto.
- La regla evita frases genericas como "seguir buenas practicas".
