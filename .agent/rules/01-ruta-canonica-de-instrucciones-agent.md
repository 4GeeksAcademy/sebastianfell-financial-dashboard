# Regla: Ruta canonica de instrucciones agent

## Proposito
Eliminar confusion entre rutas de configuracion de agentes.

## Alcance
Archivos de guia operativa y onboarding de agentes.

## Disparadores
- Se modifica AGENTS.md.
- Se agrega documentacion para agentes.
- Se detecta referencia a .agents.

## Fuera de alcance
- Rutas de codigo frontend/backend.
- Variables de entorno de runtime.

## Checklist verificable
- La ruta oficial de reglas es ./.agent/rules.
- La ruta oficial de skills es ./.agent/skills.
- Si aparece ./.agents en documentacion, se corrige.
- Se valida existencia de .agent/rules y memory-bank al iniciar trabajo.
